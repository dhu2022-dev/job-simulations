# Job Simulations Portfolio

This repository contains various job simulations and virtual experience programs completed to build practical skills in software engineering, quantitative analysis, and related fields. These projects simulate real-world tasks from companies like J.P. Morgan Chase and Goldman Sachs.

## Overview

These simulations provide hands-on experience with industry-standard technologies and practices, demonstrating proficiency in:

- **Backend Development**: Spring Boot, Kafka, REST APIs, database design
- **Data Science & Quantitative Finance**: Time series forecasting, financial modeling, statistical analysis
- **Software Engineering**: Clean code practices, testing, system architecture
