## Program Overview

This program provides practical understanding of Agile methodologies through educational content and quizzes. It covers core Agile concepts, Scrum framework, and how these practices are implemented in large software engineering organizations.

**Program Structure**: Each task/section covers a different aspect of Agile methodologies with educational content followed by quizzes to reinforce learning.

## Tasks Overview

1. **[Introduction to Agile](./task1-intro-to-agile)** - Core Agile principles and values
2. **[Scrum at JPMC](./task2-scrum-at-jpmc)** - Scrum framework implementation at JPMC
3. **[User Stories](./task3-user-stories)** - Writing and managing user stories
4. **[Backlog Refinement](./task4-backlog-refinement)** - Backlog management and prioritization
5. **[Daily Standups](./task5-daily-standups)** - Daily Scrum practices
6. **[Sprint Reviews and Retrospectives](./task6-sprint-reviews-retrospectives)** - Sprint ceremonies

## Key Learnings

This program covers:
- Agile Manifesto principles and values
- Scrum framework roles, ceremonies, and artifacts
- User story writing and acceptance criteria
- Backlog refinement techniques
- Effective daily standup practices
- Sprint review and retrospective best practices
- How Agile is applied in large-scale software engineering organizations
